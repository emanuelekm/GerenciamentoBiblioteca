# Sistema de Biblioteca

## 📌 Descrição
Sistema desenvolvido em C# com Windows Forms para gerenciamento de biblioteca.

## ⚙️ Funcionalidades
- Login com níveis de acesso
- Cadastro de livros
- Empréstimo e devolução
- Controle de status

## 🛠️ Tecnologias
- C#
- Windows Forms
- MySQL

## ▶️ Como executar
1. Clone o repositório
2. Configure o banco de dados
3. Execute o projeto

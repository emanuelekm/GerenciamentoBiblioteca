## Descrição

Este projeto consiste no desenvolvimento de um sistema desktop para gerenciamento de biblioteca, com foco no controle de usuários, acervo e empréstimos.

O problema abordado é a dificuldade de organizar manualmente livros, usuários e registros de empréstimos, o que pode gerar inconsistências e perda de controle. Como solução, o sistema oferece uma interface intuitiva integrada a um banco de dados, permitindo o gerenciamento completo das operações de uma biblioteca.

O impacto do sistema está na automatização dos processos, melhoria na organização e maior controle sobre empréstimos, devoluções e usuários, sendo aplicável em bibliotecas de pequeno e médio porte.

---

## Objetivo

* Desenvolver um sistema completo com autenticação de usuários
* Aplicar conceitos de CRUD com banco de dados MySQL
* Trabalhar com diferentes níveis de acesso (Administrador e Leitor)
* Praticar desenvolvimento de aplicações desktop com Windows Forms
* Simular um sistema real de gerenciamento

## Como executar

```bash
# Clone o repositório
git clone https://github.com/emanuelekm/GerenciamentoBiblioteca.git
```

### Configuração do banco de dados

1. Crie um banco de dados no MySQL
2. Execute o script SQL disponível no projeto (se houver)
3. Configure a string de conexão no código

### Executar o sistema

* Abra o projeto no Visual Studio
* Compile a solução
* Execute o sistema (F5)

---

## Funcionalidades

* Cadastro e login de usuários
* Diferenciação de permissões (Administrador / Leitor)
* Cadastro, edição e remoção de livros
* Sistema de empréstimo e devolução
* Controle de status (Em andamento, Devolvido, Em atraso)
* Exibição de dados em DataGridView
* Gerenciamento de acervo completo
